Prop Firm Challenge Readiness Analyzer

AI-powered evaluation matching rules against historical performance data

How It Works

Prop firms impose strict limits on daily drawdown, overall loss, and trading activity. This application acts as a sandbox, scanning your trade logs (CSV) for subtle warning signs before you invest in an evaluation.

  • Normalized Parsing: Seamlessly imports raw CSV logs from MT4, MT5, or cTrader.
  • Hard Rule Evaluator: Identifies potential breach triggers on daily loss and maximum drawdown.
  • Machine Learning Scoring: Predicts passing probability using a classification model trained on disciplined trader statistics.

Key Features

  • Behavioral Analysis: Flags overtrading (>8 trades/day) and revenge trading (oversized sizing after losses).
  • Equity Visualization: Renders your cumulative P&L curve chronologically.
  • Presets Engine: Loaded with standard 2-step (FTMO-style) and aggressive 1-step (Topstep-style) presets.
